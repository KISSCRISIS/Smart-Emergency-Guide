# 🏥 Smart Emergency Guide
**Learn Faster. Think Smarter. Save Lives.**

## 🚀 Overview
Smart Emergency Guide is a comprehensive Emergency Medicine learning platform designed for medical students, residents, and healthcare professionals. From ECG interpretation to critical care management, everything is consolidated in one AI-powered learning environment.

### 📚 Core Content Areas
- **Volume I:** ECG Atlas (40 chapters)
- **Volume II:** POCUS Atlas (40 chapters)
- **Volume III:** EM Drug Handbook (100+ chapters)
- **Volume IV:** Critical Care Infusion Handbook
- **Volume V:** Toxicology Master Tables
- **Volume VI:** EM Board Master Review

### ✨ Key Features
- ✅ Complete ECG and POCUS pattern recognition
- ✅ Comprehensive drug database with dosing
- ✅ Spaced-repetition flashcards
- ✅ Board-style quiz engine with test simulation
- ✅ AI-powered study assistant
- ✅ Arabic-enhanced content with bilingual support
- ✅ Progress tracking and analytics
- ✅ Badges and achievement system
- ✅ Content protection with watermarking
- ✅ Admin approval workflow

## 🚦 Quick Start

### 1. Clone the Repository
```bash
git clone https://github.com/yourusername/smart-emergency-guide.git
cd smart-emergency-guide
```

### 2. Install dependencies
```bash
npm install
```

### 3. Configure environment
```bash
cp .env.example .env
```

### 4. Run dev server
```bash
npm run dev
```
