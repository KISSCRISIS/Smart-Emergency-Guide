# SEG Intern Page Template

Use this template for Intern Doctors Path module pages.

## Style

- Use the existing `InternalPage` wrapper.
- Use `GlassCard` cards.
- Match the visual style of existing intern pages.
- Keep content concise, practical, and scannable.
- Prefer cards, checklists, short grids, and exam traps.

## Language

- English is the main medical language.
- Arabic appears only as short pearls/golden rules.
- Avoid long Arabic translation of medical content.

## Standard Page Structure

1. Short title and subtitle.
2. One-line educational disclaimer when the page has clinical workflow content.
3. 6–9 focused content sections.
4. Practical intern tasks.
5. Red flags or escalation points when relevant.
6. Communication or documentation points when relevant.
7. Quick checklist when useful.
8. JMC / exam traps.
9. Short final practical takeaway.

## Recommended Section Types

- What this page is for.
- What to do first.
- What to check.
- When to call senior.
- How to document.
- How to communicate.
- What not to miss.
- Exam traps.
- Quick checklist.

## Avoid

- Medication doses on general workflow pages.
- Fixed triage times.
- Universal triage color protocols.
- Hospital-specific rules.
- Long textbook explanations.
- Overconfident clinical instructions.

## Every Page Should Answer

- What should an intern notice?
- What should an intern do safely?
- When should an intern escalate?
- What is the common exam trap?
