# SEG Master QA + Integration Blueprint

This package is the planning layer for integrating the seven approved SEG content baselines into the website.

## Included files

- `MASTER_QA_REPORT.md` — QA findings and go/no-go decision
- `MASTER_QA_SUMMARY.csv` — pack-level metrics
- `QA_ISSUES.csv` — issue/polish flags
- `CONTENT_INVENTORY.csv` — all indexed topics across all packs
- `ROUTE_PLAN.csv` — route prefixes and import plan
- `IMAGE_BRIEF_MASTER.csv` — combined image brief records where available
- `FINAL_VERIFICATION_BACKLOG.csv` — medical/protocol verification tasks
- `SITE_INTEGRATION_BLUEPRINT.md` — website integration architecture
- `TOPIC_PAGE_SCHEMA.md` — unified topic frontmatter and required content blocks
- `MASTER_SIDEBAR_STRUCTURE.json` — proposed sidebar plan
- `segContentRegistry.ts` — TypeScript registry skeleton
- `INTEGRATION_TASKS.md` — practical next steps
- `ECG_ROUTE_NAMESPACE_NOTE.md` — ECG duplicate flat slug routing note

## Decision

The content is ready for integration planning. It is not yet a final public medical protocol layer.
