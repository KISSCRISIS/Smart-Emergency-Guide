# ECG Content Collection Checklist

Create this folder on phone/cloud:

SEG_ECG_READY

Inside it, create:

1. `ecg_content`
2. `ecg_images`
3. `ecg_notes`
4. `ecg_sources`
5. `ecg_review`

## Put in `ecg_content`

- your complete ECG notes
- any previous ECG lecture files
- ECG topic lists
- JMC ECG questions/traps if available
- your preferred ECG sequence

## Put in `ecg_images`

Only ECG images/strips.

For every image, add a note:
- what topic it belongs to
- where it came from
- whether it is safe to use
- whether diagnosis is verified

## Put in `ecg_notes`

- exam traps
- Arabic pearls
- must-include patterns
- things to avoid
- known app image mismatch issues

## Put in `ecg_sources`

- open-access ECG sources if available
- citations/links
- permissions notes
- screenshots only if legally usable

## Put in `ecg_review`

- medical reviewer notes later
- image verification table
- correction log

## Minimum before implementation

Do not implement ECG until you have at least:

- [ ] topic list approved
- [ ] page structure approved
- [ ] no mismatched visuals
- [ ] image policy approved
