# SEG Content Safety Checklist

Before committing any clinical module, confirm:

- Build passed.
- Browser verification passed.
- Disclaimer present.
- No medication doses.
- No fixed triage waiting times.
- No hospital-specific protocols.
- Arabic pearls are short and readable.
- No copied copyrighted text/questions.
- Source note included when appropriate.
- Commit only target file.
- No git add .
- No push.

Drug scan:
mg, mcg, mL, units/kg, dose, dosage, bolus, infusion, drip, protocol, RSI, antidote

Toxicology scan:
mg, mcg, mL, units/kg, dose, dosage, bolus, infusion, drip, protocol, activated charcoal, antidote, decontamination

Emergency Cases scan:
mg, mcg, mL, units/kg, dose, dosage, bolus, infusion, drip, protocol, give, administer, discharge
