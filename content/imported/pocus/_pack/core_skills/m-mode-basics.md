# M-Mode Basics

## Definition
M-mode tracks motion along a single ultrasound line over time.

## Why it matters in the ED
Useful for lung sliding, fetal cardiac activity, and some cardiac motion documentation.

## Probe / Patient / Acquisition
Place the line through the structure that should move.

## Key Findings to Look For
- Start with the focused clinical question.
- Confirm the relevant anatomy before interpreting pathology.
- Compare sides or views when appropriate.
- Save representative images or clips for QA when the scan is used clinically.

## Common ED Trap
Using M-mode as a substitute for understanding the 2D image.
