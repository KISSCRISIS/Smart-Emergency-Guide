# Medical Students Path QA Checklist

## Before implementation

- [ ] Folder is separate from Intern/GP/ECG.
- [ ] Target route confirmed.
- [ ] Does not claim to be official curriculum.
- [ ] Sources used conceptually only.
- [ ] No copyrighted text copied.
- [ ] No textbook images included.

## Content QA

- [ ] Student role is clear.
- [ ] Supervision is clear.
- [ ] Preclinical/clinical/senior stages included.
- [ ] Jordan context included without overclaiming.
- [ ] Red flags included.
- [ ] OSCE and case presentation included.
- [ ] Internship bridge included.
- [ ] Arabic pearls short.
- [ ] No generic filler.
- [ ] No doses.
- [ ] No fixed triage times.
- [ ] No local protocols.
- [ ] No independent treatment/discharge advice.

## Build/route QA

- [ ] npm run build passes.
- [ ] Student hub loads.
- [ ] Module cards visible.
- [ ] Mobile layout acceptable.
- [ ] Desktop layout acceptable.
- [ ] Links do not point to missing routes unless disabled/coming soon.

## Git QA

- [ ] git status --short reviewed.
- [ ] git diff --stat reviewed.
- [ ] Only student target file committed.
- [ ] No git add .
- [ ] No push unless explicitly requested.
