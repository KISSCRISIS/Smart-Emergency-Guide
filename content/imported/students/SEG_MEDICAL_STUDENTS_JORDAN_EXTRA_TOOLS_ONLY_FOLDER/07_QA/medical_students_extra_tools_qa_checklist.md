# Medical Students Extra Tools QA Checklist

## Practice Questions

- [ ] Every question labeled original SEG practice question.
- [ ] No copied wording from source PDFs or university materials.
- [ ] No official-exam claim.
- [ ] Answer key present.
- [ ] Explanations short.
- [ ] Focus on student role, red flags, OSCE, supervised reasoning.
- [ ] No medication doses.
- [ ] No local protocols.

## OSCE Stations

- [ ] Original SEG stations.
- [ ] Student role clear.
- [ ] Consent/professionalism included.
- [ ] Red flags included.
- [ ] No treatment protocols.
- [ ] No doses.

## Logbooks / Checklists

- [ ] No patient identifiers.
- [ ] Reflection prompts included.
- [ ] Jordan/MOH/community exposure represented.
- [ ] ED observation framed as observation, not independent management.

## Case Presentation

- [ ] One-minute presentation template.
- [ ] ISBAR included.
- [ ] Red-flag reporting phrase included.
- [ ] Uncertainty/senior review phrase included.

## UI Copy

- [ ] Medical Students path separate from Intern and GP.
- [ ] No fake active buttons.
- [ ] No official curriculum claim.
- [ ] Disclaimer included.
