# SEG MEDICAL STUDENTS JORDAN — EXTRA TOOLS ONLY

This folder contains ONLY extra tools for the Medical Students Jordan Path.

It is separate from:
- Intern Path
- GP Path
- ECG Essentials
- Resident Path
- EMS Path

Purpose:
Add practical tools for Jordanian medical students:
1. Original student-level practice questions
2. OSCE station templates
3. clerkship/rotation checklists
4. ED observation logbook
5. case presentation templates
6. UI copy
7. Claude prompts and QA

Important:
The practice questions and OSCE stations are ORIGINAL SEG educational content.
They are NOT official university questions.
They are NOT copied from Rosh, Rosen, Tintinalli, MOH, or any Jordanian university.

Jordan alignment:
This pack is designed to fit broadly with Jordanian public medical education and Ministry of Health / public clinical training environments without claiming to be an official curriculum.

Core source basis:
- Tintinalli’s Emergency Medicine Examination and Board Review 2023
- Rosen’s Emergency Medicine 10e Q&A
- Rosh Rapid Review — Emergency Medicine Rotation Exam
- official Jordanian public health/medical education context used conceptually

Rules:
- no medication doses
- no independent treatment plans
- no local protocols
- no copied source questions
- no claim of official curriculum
