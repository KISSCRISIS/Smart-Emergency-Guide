## SEG rules used
- Short independent pages, not one-page stacking.
- Official academic English for scientific content.
- Arabic simplification for analogies and reasoning.
- Board-style assessment question on each page.
- No final numeric dosing/protocol layer.
- Final visuals and site styling deferred to the website phase.
