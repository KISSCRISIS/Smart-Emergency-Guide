# QA Checklist

## Structure
- [ ] Every topic is a standalone markdown file.
- [ ] `airway_index.json` matches all markdown files.
- [ ] `airway_routes.ts` maps cleanly to the website route plan.
- [ ] No one-page stacking.

## Language
- [ ] English is used for official academic content.
- [ ] Arabic is used for simplification and reasoning.
- [ ] Assessment questions are in English.

## Medical safety
- [ ] Every dose/setting/algorithm draft is labeled educational and pending local verification.
- [ ] No unsupported hospital-specific claim.
- [ ] No fixed timing/disposition rules without protocol review.
- [ ] Ventilator settings are drafts, not orders.
- [ ] Failed airway pages are reviewed by airway lead before publication.

## Visuals
- [ ] Every page has an image brief.
- [ ] Images are original, licensed, or verified open-use.
- [ ] No copyrighted textbook images are embedded without permission.
