# SEG ECG Essentials — Scientific Content Pack

This pack contains ready-to-import educational material for **Smart Emergency Guide (SEG)** ECG Essentials.

## Design Decision
Do **not** stack ECG content in one giant page. Each topic is a short, independent, strong page.

## Included Sections
1. `content/ecg/beginner-roadmap/` — 9 beginner roadmap pages.
2. `content/ecg/never-miss/` — 8 catastrophic ECG patterns that must not be missed.
3. `content/ecg/atlas/` — 34 independent ECG atlas pattern pages.
4. `content/ecg/practice/` — MCQ bank, mini cases, and rapid prompts.
5. `assets/image-briefs/` — visual matching instructions for future ECG images.
6. `data/ecg-content-index.json` — route map and metadata for site integration.

## Content System
- Main academic content: English.
- Simplification/analogy: Arabic.
- Each concept includes: definition, ECG clues, ED importance, common trap, Arabic simplification, image brief, assessment question, answer, Arabic rationale.

## Safety Rules
- Educational only; not a substitute for bedside clinical judgment.
- No medication doses are embedded in this pack.
- Emergency actions are intentionally written as high-level escalation instructions and must be aligned with local protocols.

## Suggested SEG Routes
- `/ecg/beginner-roadmap/[slug]`
- `/ecg/never-miss/[slug]`
- `/ecg/atlas/[slug]`
- `/ecg/practice/[slug]`
