# EMS / Paramedics Path Safety Rules

## Core Rule

EMS Path is for prehospital recognition, scene safety, and handover education.

It is not a treatment protocol.

## Must include

- disclaimer
- scene safety
- role boundaries
- follow local protocol/medical direction
- red flags
- handover
- documentation
- no local operational claims

## Must avoid

- medication doses
- medication administration instructions
- procedure instructions
- specific transport destination rules
- local ambulance dispatch policy
- fixed triage times
- advanced airway steps
- defibrillation/cardioversion instructions
- copied source text/questions
- claiming official EMS protocol

## Scan terms

Search for:
- mg
- mcg
- mL
- units/kg
- dose
- dosage
- bolus
- infusion
- drip
- administer
- give
- intubate
- defibrillate
- cardiovert
- protocol
- within 10 minutes
- within 30 minutes
- transport to
- bypass

Context review is required:
Some terms may appear in warnings/disclaimers. Read context before failing.
