# EMS / Paramedics Path — Identity

## Path Name

EMS / Paramedics Path  
إسعاف وطوارئ ما قبل المستشفى

## Main Purpose

Help EMS learners and prehospital providers think safely about:
- scene safety
- first impression
- life threats
- ABCDE recognition
- transport priority
- communication with receiving ED
- handover
- documentation
- role boundaries

## What this path teaches

An EMS/prehospital learner should learn:

- how to approach a scene safely
- how to identify an unstable patient early
- how to organize ABCDE thinking
- how to recognize high-risk chest pain, dyspnea, stroke, trauma, seizure, overdose, pediatrics, and pregnancy emergencies
- how to communicate clearly with ED
- how to document prehospital observations
- how to avoid unsafe delay

## What this path does NOT teach

- medication doses
- local ambulance protocols
- independent advanced procedures
- transport rules for a specific country/service
- replacing certified EMS training
- replacing medical direction or dispatch policy
- replacing ED/hospital clinical judgment

## Golden Rule

EMS Path = scene safety first, patient danger second, rapid communication always.

## Arabic Pearl
