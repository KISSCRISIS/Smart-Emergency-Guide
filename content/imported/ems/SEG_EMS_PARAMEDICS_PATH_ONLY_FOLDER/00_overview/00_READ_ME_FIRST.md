# SEG EMS / Paramedics Path ONLY — READ ME FIRST

This folder contains ONLY EMS / Paramedics Path content.

It is intentionally separated from:
- Intern Path
- GP Path
- Medical Students Path
- ECG Essentials
- Resident Path
- Drug Handbook
- Antidotes Handbook

Purpose:
Create a prehospital emergency recognition, stabilization awareness, and handover pathway for EMS/paramedics.

Core identity:
EMS / Paramedics Path = scene safety, first impression, ABCDE recognition, prehospital prioritization, and safe handover.
NOT an ambulance protocol.
NOT medication dosing.
NOT local Civil Defense / EMS operational policy.
NOT a substitute for certified paramedic training.

Core uploaded emergency medicine source basis:
1. Tintinalli’s Emergency Medicine Examination and Board Review 2023
2. Rosen’s Emergency Medicine 10e Q&A
3. Rosh Rapid Review — Emergency Medicine Rotation Exam

Use source-informed synthesis only.
Do not copy copyrighted wording, questions, tables, images, or explanations.

Recommended implementation timing:
Do NOT implement before Intern MVP is stable unless explicitly decided.
When implementation starts, begin with one EMS hub page first, then expand into subpages later.
