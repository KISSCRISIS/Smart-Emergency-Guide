# EMS / Paramedics Path Style Guide

## Language

Main UI/content:
English

Arabic:
Short pearls only.

## Tone

- prehospital
- safety-focused
- fast recognition
- operationally aware
- non-protocol
- handover-oriented

## Page structure per module

Each module should include:

1. Purpose
2. Prehospital danger signs
3. What EMS should notice
4. What to communicate
5. What not to delay for
6. Common prehospital trap
7. Arabic pearl

## Preferred wording

Use:
- recognize
- escalate
- notify receiving ED
- follow local protocol/medical direction
- transport priority according to local policy
- safety first
- prehospital red flag
- handover

Avoid:
- administer
- dose
- protocol steps
- definitive treatment plan
- intubate/do procedure without context
- local operational commands
