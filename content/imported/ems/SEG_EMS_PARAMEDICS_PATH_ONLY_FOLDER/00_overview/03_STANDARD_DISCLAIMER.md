# EMS / Paramedics Path Standard Disclaimer

Use this disclaimer near the top of EMS pages:

General educational framework for EMS/prehospital learning. This pathway is not an ambulance protocol and does not replace certified EMS training, local protocols, medical direction, dispatch policy, or receiving-hospital guidance.

## Short card disclaimer

Educational guide only — not an EMS protocol.

## When to repeat it

Repeat or show prominently in modules involving:
- airway/breathing
- shock
- chest pain
- stroke
- trauma
- pediatrics
- pregnancy
- overdose/toxicology
- mass casualty
- procedures/medications
