# Shortness of Breath Prehospital Approach

## Purpose

Help EMS approach dyspnea using danger recognition.

## Red Flags

- inability to speak
- cyanosis
- exhaustion
- confusion/drowsiness
- silent chest
- chest pain
- syncope
- suspected anaphylaxis
- suspected PE
- suspected pneumothorax
- severe wheeze
- pulmonary edema features
- fever/toxic appearance

## What EMS Should Notice

- position
- speech
- work of breathing
- accessory muscle use
- chest symmetry
- wheeze/crackles/silent chest
- edema/heart failure clues
- allergen/exposure clues
- trauma clues

## What To Communicate

- severity
- suspected cause
- oxygenation/ventilation concerns if available
- mental status
- deterioration or improvement
- interventions under protocol

## Common Prehospital Trap

Treating all wheeze as asthma.
