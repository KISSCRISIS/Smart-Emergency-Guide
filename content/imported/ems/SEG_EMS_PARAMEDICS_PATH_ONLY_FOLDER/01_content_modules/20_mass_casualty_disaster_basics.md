# Mass Casualty / Disaster Basics

## Purpose

Give EMS learners a non-protocol overview of disaster thinking.

## Key Concepts

- scene safety first
- incident command according to local system
- communication discipline
- resource awareness
- triage according to local policy/training
- do not self-deploy outside system
- identify hazards
- protect responders
- distribute patients according to system direction
- documentation and accountability

## What EMS Should Learn

- one rescuer cannot manage the scene alone
- communication is treatment
- triage systems are local/protocol-driven
- personal safety and PPE matter
- transport decisions depend on system coordination

## Common Prehospital Trap

Trying to help without integrating into command structure.
