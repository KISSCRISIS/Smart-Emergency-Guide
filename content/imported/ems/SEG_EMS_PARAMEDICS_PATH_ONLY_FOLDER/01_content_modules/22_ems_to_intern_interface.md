# EMS-to-Intern Interface

## Purpose

Help EMS and ED interns understand each other’s needs.

## What EMS Brings

- scene story
- timeline
- first impression
- patient trend
- mechanism
- bystander information
- home medications/containers
- ECG or vitals if available
- response to protocol-based actions
- safety concerns

## What Interns Need

- onset/last known well
- instability signs
- red flags
- what changed
- what was done
- what was found at scene
- medications/allergies if known
- family/bystander contact information if available

## Best Interface

EMS gives a structured handover.  
Intern listens, clarifies critical missing data, and escalates if unstable.

## Common Trap

ED team ignores EMS timeline; EMS handover becomes too vague.
