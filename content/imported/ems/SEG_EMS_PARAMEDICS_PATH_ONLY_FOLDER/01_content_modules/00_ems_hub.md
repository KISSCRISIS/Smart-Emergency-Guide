# EMS / Paramedics Path Hub

**Title:** EMS / Paramedics Path  
**Subtitle:** Prehospital recognition, scene safety, stabilization awareness, and ED handover  
**Arabic subtitle:** مسار الإسعاف والطوارئ قبل الوصول للمستشفى

**Disclaimer:**  
General educational framework for EMS/prehospital learning. This pathway is not an ambulance protocol and does not replace certified EMS training, local protocols, medical direction, dispatch policy, or receiving-hospital guidance.

## Path Purpose

This pathway helps EMS learners and prehospital providers build a safe emergency thinking structure:

- scene safety
- first impression
- ABCDE recognition
- life threats
- transport priority thinking
- receiving ED notification
- handover
- documentation
