# Emergency Resident / Specialist Path Hub

**Title:** Emergency Resident / Specialist Path  
**Subtitle:** Advanced ED reasoning, resuscitation leadership, and disposition decisions  
**Arabic subtitle:** مسار مقيم/أخصائي الطوارئ: قيادة الإنعاش، التفكير المتقدم، وقرارات التصريف

**Disclaimer:**  
General educational framework for emergency medicine residents and specialists. This pathway is not a hospital protocol and does not replace local guidelines, consultant supervision, institutional policies, board-approved references, or patient-specific clinical judgment.

## Path Purpose

This pathway is for the ED doctor who must move beyond recognizing danger into:

- leading the first minutes
- choosing the dangerous differential
- directing resources
- communicating with consultants
- supervising juniors
- deciding disposition risk
- preparing for board-style emergency medicine reasoning
