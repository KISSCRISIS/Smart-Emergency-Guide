## UI Rules

- Keep Resident visually more advanced than Intern path.
- Use decision-card design: danger differential / decision point / disposition / trap.
- Do not use student/intern language as primary tone.
- Buttons should say Analyze / Lead / Decide / Consult / Review.
- Use badges: Critical, Disposition, Consultant, Board Trap, Reassess, High-Risk.
- No fake functional buttons.
