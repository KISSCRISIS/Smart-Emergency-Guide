# Resident / Specialist Path Safety Rules

## Core Rule

Resident Path teaches advanced ED reasoning and decision frameworks, not hospital protocols.

## Must include

- disclaimer
- local protocol/supervision boundaries
- disposition risk thinking
- consultant communication
- cognitive traps
- exam traps
- reassessment language
- no copied source questions

## Must avoid

- medication doses
- medication recipes
- exact ventilator settings
- procedural step-by-step instructions
- local hospital pathways as universal rules
- fixed triage times
- cath lab/stroke/sepsis bundle claims unless framed as local protocol dependent
- independent procedural manual language
- copied source text/questions/tables/images

## Scan terms

Search for:
- mg
- mcg
- mL
- units/kg
- dose
- dosage
- bolus
- infusion
- drip
- administer
- give
- intubate
- ventilator settings
- protocol
- within 10 minutes
- within 30 minutes
- discharge home
- safe to discharge
- must always

Context review is required:
Some terms may appear in warnings/disclaimers. Read context before failing.
