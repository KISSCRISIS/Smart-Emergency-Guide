# SEG EMERGENCY RESIDENT / SPECIALIST PATH ONLY — READ ME FIRST

This folder contains ONLY Emergency Resident / Emergency Specialist Path content.

It is intentionally separated from:
- Intern Path
- GP Path
- Medical Students Path
- EMS / Paramedics Path
- ECG Essentials
- Drug Handbook
- Antidotes Handbook

Purpose:
Create a resident-level emergency medicine pathway focused on:
- leadership in the ED
- differential diagnosis under uncertainty
- resuscitation priorities
- disposition and risk stratification
- supervision of junior doctors
- procedural decision-making
- high-yield emergency medicine exam thinking
- safe escalation and consultation

Core identity:
Emergency Resident / Specialist Path = advanced ED reasoning, resuscitation leadership, disposition thinking, and exam-focused emergency decision-making.
NOT a hospital protocol.
NOT a drug dose handbook.
NOT a replacement for consultant supervision, local policies, or board-approved guidelines.

Core uploaded source basis:
1. Tintinalli’s Emergency Medicine Examination and Board Review 2023
2. Rosen’s Emergency Medicine 10e Q&A
3. Rosh Rapid Review — Emergency Medicine Rotation Exam

Use source-informed synthesis only.
Do not copy copyrighted wording, questions, tables, images, or explanations.

Recommended implementation timing:
Do NOT implement before Intern MVP is stable unless explicitly decided.
When implementation starts, begin with one Resident/Specialist hub page first, then expand into subpages later.
