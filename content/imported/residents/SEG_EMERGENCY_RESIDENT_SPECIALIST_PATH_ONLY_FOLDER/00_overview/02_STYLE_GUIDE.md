# Resident / Specialist Path Style Guide

## Language

Main UI/content:
English

Arabic:
Short pearls only.

## Tone

- advanced
- decisive but safe
- resuscitation-oriented
- exam-focused
- disposition-aware
- consultant-communication aware

## Page structure per module

Each module should include:

1. Purpose
2. Why it matters in ED
3. Resident-level first-pass thinking
4. Dangerous diagnoses not to miss
5. Decision points
6. Disposition implications
7. Consultation triggers
8. Cognitive traps
9. Exam traps
10. Arabic pearl

## Preferred wording

Use:
- resuscitation priority
- disposition risk
- cognitive trap
- senior/consultant discussion
- local protocol
- high-risk differential
- reassessment
- risk stratification
- ED decision point
- board trap

Avoid:
- exact medication doses
- rigid local protocols
- one-size-fits-all discharge rules
- unsupported “always/never”
- copied copyrighted question wording
