# Emergency Resident / Specialist Path — Identity

## Path Name

Emergency Resident / Specialist Path  
مقيم أو أخصائي طب طوارئ

## Main Purpose

Help emergency residents and specialists build structured, high-yield ED reasoning:

- leading early resuscitation
- prioritizing life threats
- creating a differential under uncertainty
- deciding who needs ICU/ward/OR/cath lab/stroke pathway/specialist consultation
- supervising interns and students safely
- using ECG, POCUS, labs, and imaging in context
- recognizing exam traps
- communicating disposition risk clearly

## Target users

- Emergency medicine residents
- Emergency medicine specialists
- Physicians preparing for emergency medicine board-style exams
- Senior ED doctors supervising juniors
- Clinicians who need advanced ED reasoning frameworks

## What this path teaches

- ED leadership mindset
- resuscitation priorities
- diagnostic uncertainty
- high-risk differentials
- disposition thinking
- procedural risk assessment
- consultant communication
- cognitive error prevention
- exam traps and board-style reasoning

## What this path does NOT teach

- local hospital protocols
- medication dosing tables
- independent procedural manuals
- replacing consultant supervision
- replacing national/institutional guidelines
- copied board questions
- definitive treatment algorithms without local adaptation

## Golden Rule

Resident Path = think broadly, act safely, lead clearly, reassess constantly, and own disposition risk.

## Arabic Pearl

المقيم في الطوارئ لا يبحث عن التشخيص فقط؛ يبحث عن الخطر، القرار، ومن يحتاج إنقاذًا الآن.
