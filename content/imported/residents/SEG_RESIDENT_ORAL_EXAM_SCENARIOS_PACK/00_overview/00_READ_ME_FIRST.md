# SEG Resident Oral Exam Scenarios Pack

This folder contains ORIGINAL emergency medicine oral-exam style scenarios for the Smart Emergency Guide (SEG) Resident / Specialist Path.

## Purpose

Prepare emergency residents and specialists for case-based oral examination thinking:
- first critical actions
- ABCDE and resuscitation priorities
- dangerous differential diagnosis
- data request strategy
- evolving case response
- consultation and disposition
- critical failure avoidance

## Core Source Basis

This pack is source-informed by the three SEG core references:
1. Tintinalli’s Emergency Medicine Examination and Board Review 2023
2. Rosen’s Emergency Medicine 10e Q&A
3. Rosh Rapid Review — Emergency Medicine Rotation Exam

Use is conceptual only:
- topic coverage
- emergency reasoning
- high-yield red flags
- common traps
- disposition logic
