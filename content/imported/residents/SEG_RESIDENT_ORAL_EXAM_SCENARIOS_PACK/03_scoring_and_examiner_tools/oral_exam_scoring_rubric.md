# Oral Exam Scoring Rubric

Score each domain 0–2.

## 1. Initial Stabilization

0 = jumps to diagnosis/test without stability assessment  
1 = mentions ABCDE but incomplete  
2 = clear ABCDE, monitors/access/help, immediate danger recognition

## 2. Differential Diagnosis

0 = narrow or misses life threats  
1 = includes some dangerous diagnoses  
2 = organized differential with dangerous/likely/mimics

## 3. Data Strategy

0 = random tests or delays critical care  
1 = asks reasonable but incomplete data  
2 = targeted vitals/history/exam/ECG/labs/imaging based on scenario

## 4. Case Evolution

0 = does not adapt to new information  
1 = partially updates plan  
2 = reassesses and changes priorities appropriately

## 5. Consultation / Disposition

0 = no disposition or unsafe disposition  
1 = disposition mentioned but weak reasoning  
2 = clear consultant/disposition/risk reasoning

## 6. Communication

0 = disorganized or unsafe language  
1 = understandable but not concise  
2 = structured, calm, safe, closed-loop style

## Critical Failure Override

Any critical failure can convert the station to fail even if points are otherwise adequate.
