## Integration Rule
Do not stack cases on one page. Each case must remain a standalone page with:
- clinical vignette;
- first 60-second impression;
- what can kill now;
- key clues;
- first ED actions conceptually;
- essential tests;
- common trap;
- Arabic simplification;
- assessment question.

## Safety Rule
This package intentionally avoids fixed drug doses, infusion rates, procedure timing promises, and rigid disposition rules. Final site publication requires medical audit and local-protocol review.
