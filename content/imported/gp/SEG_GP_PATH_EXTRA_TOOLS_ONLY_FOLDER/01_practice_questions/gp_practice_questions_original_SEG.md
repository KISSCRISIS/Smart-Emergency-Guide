# GP Path Practice Questions — Original SEG Set

**Label:** Practice question — original SEG educational question, not an official JMC/Rosh/Rosen/Tintinalli question.

**Purpose:**  
Train GP-level emergency recognition, red flags, referral decisions, and safety-netting.

**Rules:**  
- No medication doses.
- No local protocols.
- No fixed triage times.
- No copied source questions.
- Focus on recognition and referral, not ED management.

---

## Question 1 — Chest Pain

**Practice question — original SEG educational question, not an official exam question.**

A 61-year-old man presents to a primary care clinic with central chest pressure, sweating, and shortness of breath. The pain started while walking and has not fully resolved. What is the safest GP action?

A. Reassure and review routinely
B. Treat as indigestion if abdominal exam is normal
C. Arrange urgent ED assessment and communicate chest pain red flags
D. Ask him to return tomorrow if symptoms persist

**Answer:** C

**Explanation:** Chest pain with sweating, exertional onset, and dyspnea is high-risk. The GP role is urgent recognition, referral, and clear handover.
