# GP Path Safety Rules

## Core Rule

GP Path is Emergency Recognition & Referral, not ED treatment.

## Must include

- disclaimer
- red flags
- urgent referral triggers
- what to communicate to ED
- safety-netting
- documentation prompts
- Arabic pearls only

## Must avoid

- medication doses
- drug administration instructions
- local hospital protocols
- fixed triage waiting times
- pediatric dosing
- antidote dosing
- decontamination procedures
- cath-lab/stroke/sepsis bundle timing
- definitive discharge rules
- copied text/questions from sources

## Scan terms

Search for:
- mg
- mcg
- mL
- units/kg
- dose
- dosage
