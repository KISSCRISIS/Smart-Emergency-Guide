# GP Path Style Guide

## Language

Main UI/content:
English

Arabic:
Short pearls only.

Do not use long Arabic translations.
Do not use generic Arabic filler.

## Tone

- Practical
- Frontline
- Safety-focused
- Red-flag oriented
- Referral-oriented

## Page structure per module

Each module should include:

1. Purpose
2. Dangerous causes to consider
3. Red flags
4. GP first-contact assessment
5. What to communicate to ED
6. Common GP traps
7. Safety-netting notes when relevant
8. Arabic pearl

## Preferred wording

Use:
- urgent ED referral
- discuss with ED/senior
- do not delay transfer
- red flags
- danger diagnosis to consider
- safety-net clearly
- not safe to reassure

Avoid:
- treat with
- give
- administer
- discharge
- safe to send home
- rule out completely
- always/never unless safety warning
- protocol
- dose/dosage
