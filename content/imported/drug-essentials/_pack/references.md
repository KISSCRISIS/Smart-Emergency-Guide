# References and Verification Sources

## Primary Board-Style Mapping Source
- Tintinalli’s Emergency Medicine Examination and Board Review, 2023. Used for topic selection, board-style framing, and emergency medicine exam relevance. Content is rewritten in original SEG educational language and is not copied verbatim.

## Dose Verification Sources for Final Layer
Final numeric dosing must be verified before public release using:
- Local ED/hospital protocol and formulary.
- DailyMed / FDA labeling for drug-specific prescribing information.
- AHA CPR/ECC guidelines for resuscitation medications and rhythm pathways.
- Specialty toxicology resources, Poison Center guidance, and EXTRIP recommendations for toxicology-linked drugs.
- Local antimicrobial stewardship guidance for antibiotics.

## Important Rule
This package is a clinical-thinking content baseline. It is not a bedside dosing formulary.
